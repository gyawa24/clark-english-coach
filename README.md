# Clark English Coach

JCI World Congress / Clark 2026 に向けた、単体HTML/CSS/JSの軽量PWAです。英語トレーニングに加えて、地域課題を聞き取り、試し、改善するためのデザイン思考フレーズと資料も扱います。ReactやNext.jsは使っていません。

## 構成

```text
/
  index.html
  manifest.json
  sw.js
  README.md
```

## Vercelでの公開手順

1. GitHubに新しいリポジトリを作成します。
2. この4ファイルをリポジトリのルートに置きます。
3. Vercelで「Add New Project」からそのリポジトリを選びます。
4. Framework Preset は「Other」または自動判定の静的サイトのままで進めます。
5. Build Command は空欄、Output Directory も空欄のままで公開できます。

個人用学習アプリなので、GitHub repository は private を推奨します。Vercel側も必要に応じて公開範囲を確認してください。

## スマホでホーム画面に追加する方法

iPhone Safari:

1. Vercelで公開したURLをSafariで開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。
4. 名前を確認して「追加」を押します。

Android Chrome:

1. Vercelで公開したURLをChromeで開きます。
2. メニューから「ホーム画面に追加」または「アプリをインストール」を選びます。
3. 表示に従って追加します。

## データ保存について

学習データはブラウザの `localStorage` に保存されます。保存される主な内容は、今日のチェックリスト、英作文、音読チェック、今日の使える1文、音読カードの進捗です。

英文読み上げはブラウザ標準の音声読み上げ機能を使います。外部の音声APIやDBには送信しません。

外部DB、API、サーバーへの送信は行いません。Vercelに公開しても、端末内の学習データはサーバーへ送られない設計です。

注意点として、同じURLでも端末やブラウザが変わると `localStorage` は共有されません。ブラウザの履歴・サイトデータを削除すると、学習データも消える場合があります。

## エクスポート/インポートによるバックアップ

アプリ内の「保存」タブからバックアップできます。

エクスポート:

1. 「保存」タブを開きます。
2. 「JSONを書き出す」を押します。
3. 作成されたJSONファイルをiCloud Drive、Google Drive、PCなどに保管します。

インポート:

1. 「保存」タブを開きます。
2. 「JSONを読み込む」を押します。
3. 以前に書き出したJSONファイルを選択します。
4. チェックリスト、英作文、音読チェック、今日の使える1文、進捗が復元されます。

この方法で、iPhoneとPCの間でも手動バックアップ・移行ができます。
